vim-mxml
================
This is a mirror from the mxml.vim file by [Abdul Qabiz](http://www.abdulqabiz.com/blog/archives/2007/10/12/vim-actionscript-and-mxml-syntax-files/)!

Usage
-----
You can just clone the repo inside your ~/.vim/bundle and then load everything up with pathogen, or use [Vundle](https://github.com/gmarik/vundle).
